## Clonar repositorio

```bash 
git clone https://github.com/YormanAlexanderCarrillo/SysoSoutPage.git
```
## Ingresar carpeta

```bash
cd SysoSoutPage
```

## Instalar paquetes necesarios

```bash
npm install
```

## Inicializar el servidor 
### Modo desarrollo

```bash
npm run dev
```
